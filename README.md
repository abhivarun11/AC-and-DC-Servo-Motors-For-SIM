This project focuses on the design and development of a motion simulator control system using 
multiple types of motors and drivers. The system integrates game telemetry data from a racing 
simulator into physical motion through stepper motors, AC servo motors, and DC motors. Initially, 
stepper motors were tested using an Arduino platform, followed by implementation of an industrial 
AC servo motor controlled via an ESP32 microcontroller. Later, a high-power DC motor driver with RS
485 communication was incorporated. Finally, the system was interfaced with a simulation game 
using motion control software to produce realistic actuator movements. A prototype motion 
platform using five stepper motors was also developed.
